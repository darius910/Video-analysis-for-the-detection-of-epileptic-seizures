Détection automatique de crises épileptiques assistée par Intelligence artificielle sur un dataset de vidéos de souris 


# Epilepsy Event Detection – Video & Signal Analysis

Ce projet vise à détecter des événements épileptiques à partir de vidéos et de signaux associés. Il combine traitement du signal, extraction de caractéristiques temporelles et fréquentielles, modèles de deep learning et machine learning pour classifier les séquences.

# Objectifs
- Prétraiter les vidéos et signaux 
- Extraire des caractéristiques  pertinentes
- Entraîner un modèle de classification 
- Évaluer la performance sur données réelles

# Méthodes utilisées
- Extraction de frames vidéo
- Analyse  des signaux
- Fusion vidéo + signal (optionnel)
- Matrice de confusion et métriques (accuracy, F1-score)

# Structure du projet
- `notebooks/`: code de prétraitement, extraction et modèles
- `results/` : courbes, exemples, visualisations
- `requirements.txt` : dépendances Python

# Résultats
- Détection fiable des événements sur les séquences annotées
- Modèle robuste aux variations inter-patients


##  Outils
Python, NumPy, SciPy, OpenCV, PyTorch / TensorFlow, Matplotlib
